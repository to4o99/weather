# weather

Това е уеб приложение за метеорологична прогноза, разработено с HTML, Tailwind CSS и Vanilla JavaScript. То извлича данни от OpenWeatherMap API, за да покаже текущото време и 3-дневна прогноза. Поддържа фиксирани локации като Свищов и Крушуна (Ловеч), както и текущата локация на потребителя чрез геолокация.
